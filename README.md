# Welcome to the HL7 Cross-Paradigm Implementation Guide: Patient Information Quality Improvement (PIQI) Framework Repository

This branch is the active branch for applying issues and trackers for the Continuous Integration (CI) Build for the HL7 Cross-Paradigm Implementation Guide: Patient Information Quality Improvement (PIQI) Framework. All issues should be logged using the official [Jira tracker system](http://hl7.org/fhir-issues) so they can be reviewed and voted upon by the HL7 Cross-Group Projects Work Group. See the [Directory of published versions](https://hl7.org/xprod/ig/uv/piqiimplementation/history.html) for a list of published editions.
